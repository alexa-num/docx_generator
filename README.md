# Генератор писем 

Приложение для автоматической генерации официальных писем в формате DOCX с использованием шаблонов, поддержкой логотипов и приложений. Разработано в рамках лабораторных работ по предмету "Функциональное и логическое программирование".

##  Возможности

- [+] Создание WPF-подобного GUI на PyQt5
- [+] Загрузка шаблона документа DOCX
- [+] Автоматическая замена плейсхолдеров в теле документа, таблицах и колонтитулах
- [+] Генерация готового письма с подстановкой пользовательских данных
- [+] Добавление одного или нескольких приложений к письму
- [+] Автоматическая нумерация приложений ("Приложение", "Приложение 1", "Приложение 2" и т.д.)
- [+] Заголовок приложения по центру, текст произвольный
- [+] Автоматическое создание перечня приложений перед подписью
- [+] Указание количества листов для каждого приложения
- [+] Валидация обязательных полей (тема письма, текст, получатель)
- [+] Группировка полей по логическим блокам (отправитель, получатель, основное, приложения)
- [+] Интерактивный предпросмотр письма в реальном времени
- [+] Поддержка логотипов (PNG, JPG, JPEG, SVG, BMP, GIF, TIFF)
- [+] Визуальный предпросмотр выбранного логотипа

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/alexa-num/docx_generator.git
cd docx_generator
```

### 2. Создание виртуального окружения

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Создание шаблона документа

Создайте файл templates/template.docx с необходимыми плейсхолдерами (см. раздел Шаблон документа)

### 5. Запуск приложения

```bash
python main.py
```

##  Использование

### Основной workflow

#### 1. Заполните информацию об отправителе:

- **Выберите логотип компании** (опционально)
  - Нажмите кнопку "Выбрать логотип"
  - Поддерживаемые форматы: PNG, JPG, JPEG, SVG, BMP, GIF, TIFF
  - Логотип автоматически масштабируется и выравнивается по левому краю

- **Введите информацию о компании**
  - Адрес
  - Телефон
  - Email
  - Другие реквизиты

- **Укажите должность и ФИО отправителя**
  - Должность (например: "Конкурсный управляющий")
  - ФИО (например: "А.В. Ростовцев")

#### 2. Заполните информацию о получателе:

- **Должность получателя**
  - Например: "Генеральному директору"

- **Название компании**
  - Например: "ООО 'Художественная школа имени Репина'"

- **ФИО/краткое имя**
  - Например: "И.К. Стрешеневу"

#### 3. Напишите письмо:

- **Тема письма** (обязательное поле)
  - Краткое содержание письма
  - Например: "О погашении задолженности"

- **Основной текст**
  - Детальное содержание письма
  - Поддерживает многострочный ввод

- **Нижний колонтитул**
  - Например: "С уважением,"

#### 4. Добавьте приложения (опционально):

1. Нажмите кнопку **"Добавить приложение"**
2. Заполните поля:
   - **Заголовок приложения** (обязательно)
   - **Текст приложения** (обязательно)
   - **Количество листов** (по умолчанию: 1)
3. Нажмите **"Добавить приложение"** для сохранения
4. Повторите шаги 1-3 для добавления нескольких приложений

#### 5. Просмотрите результат:

-  **Интерактивный предпросмотр**
  - Расположен в правой части окна
  - Обновляется автоматически при любых изменениях
  - Показывает точное содержимое будущего письма

- **Валидация в реальном времени**
  - Незаполненные обязательные поля подсвечиваются
  - Отображаются сообщения об ошибках

#### 6. Сгенерируйте документ:

1. Нажмите кнопку **"📄 Сгенерировать письмо (DOCX)"**
2. Выберите место сохранения файла
3. Укажите имя файла (по умолчанию: `generated_letter.docx`)
4. Нажмите **"Сохранить"**
5. Откройте полученный файл в Microsoft Word или совместимом редакторе

###  Управление приложениями

#### **Добавить приложение**

- **Назначение:** Создание нового приложения
- **Действие:** Сохраняет данные из полей ввода в список приложений
- **Проверки:**
  - Заголовок и текст обязательны для заполнения
  - Заголовок должен быть уникальным
- **Результат:** Приложение появляется в списке и в предпросмотре

#### **Редактировать приложение**

- **Назначение:** Изменение существующего приложения
- **Действие:** 
  1. Выберите приложение из списка (данные загрузятся в поля)
  2. Внесите необходимые изменения
  3. Нажмите "Редактировать"
- **Проверки:**
  - Приложение должно быть выбрано
  - Новый заголовок не должен дублировать другие приложения
- **Результат:** Приложение обновляется в списке и в предпросмотре

#### **Удалить приложение**

- **Назначение:** Удаление приложения из письма
- **Действие:**
  1. Выберите приложение из списка
  2. Нажмите "Удалить"
  3. Подтвердите удаление в диалоговом окне
- **Результат:** Приложение удаляется из списка и из предпросмотра
